# suken-grade1-study

このリポジトリは、実用数学技能検定 1級の合格に向けて、学習状況・弱点・過去問分析・レビュー結果を GitHub / Markdown / Issues で継続管理するための場所です。

## 目標

- 2027年春〜夏に数検1級へ合格する
- 2027年春：挑戦受験
- 2027年夏：本命受験

## 学習者の前提

- 数学科卒
- 卒業後、約8年ほど数学から離れている
- 数学の素地はあるが、計算速度・解法選択・記述答案力の復旧が必要

## 基本戦略

- 「大学数学をゼロから学ぶ」のではなく、「忘れている知識を復旧し、試験で使える状態に戻す」
- 微積・線形代数を得点源化する
- 2026年10月までに全範囲を一周する
- 2026年11月から過去問演習に入る
- 2027年春に挑戦受験し、2027年夏を本命受験とする

## GitHubで管理する理由

- 学習ログを残し、何をどれだけ進めたかを後から確認できるようにする
- 弱点をIssue化し、再演習・復習の漏れを減らす
- AIに現状共有しやすくし、進捗評価・弱点分析・次週計画を依頼しやすくする
- 過去問分析を蓄積し、得点戦略を改善する

## 最小運用ルール

- 毎日：3〜5分だけ `logs/` に学習ログを書く
- 毎週：`reviews/weekly/` に週次レビューを書く
- 毎月：`reviews/monthly/` に月次レビューを書く
- 弱点はGitHub Issue化する
- 解けなかった問題や再発しそうなミスは `mistakes/` に記録する
- 記録は学習時間の10%以下にする
- 記録に時間をかけすぎず、学習そのものを優先する

## 著作権・記録方針

- 参考書や過去問の問題文を大量に丸写ししない
- 記録するのは、教材名・ページ・問題番号・自分のミス・解法方針・再発防止策を中心にする
- AIに相談するときも、必要最小限の引用にとどめ、自分の理解や答案方針を説明する

## 主なディレクトリ

| ディレクトリ              | 用途                                          |
| ------------------------- | --------------------------------------------- |
| `logs/`                   | 日次学習ログ                                  |
| `subjects/`               | 分野別の重要トピック・典型問題・弱点整理      |
| `mistakes/`               | ミス記録                                      |
| `past-exams/`             | 過去問演習の分析                              |
| `reviews/`                | 週次・月次レビュー                            |
| `prompts/`                | AIにそのまま渡すプロンプト集                  |
| `.github/ISSUE_TEMPLATE/` | 弱点・復習・過去問レビュー用Issueテンプレート |
