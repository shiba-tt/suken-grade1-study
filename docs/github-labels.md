# GitHub Labels案

学習管理Issueを分類しやすくするためのラベル案です。

## 分野ラベル

| ラベル              | 用途                                     |
| ------------------- | ---------------------------------------- |
| `area:解析`         | 微積分、多変数解析、微分方程式など       |
| `area:線形代数`     | 行列、ベクトル空間、固有値、二次形式など |
| `area:複素解析`     | 正則性、複素積分、留数など               |
| `area:確率統計`     | 確率、期待値、分散、分布、統計など       |
| `area:整数論`       | 合同式、互除法、整数問題など             |
| `area:数値解析`     | 近似計算、数値解法など                   |
| `area:コンピュータ` | アルゴリズム、計算量など                 |
| `area:過去問`       | 過去問演習・復習・分析                   |

## 種別ラベル

| ラベル          | 用途                           |
| --------------- | ------------------------------ |
| `type:弱点`     | 苦手分野や再発するミス         |
| `type:復習`     | 再演習・復習タスク             |
| `type:過去問`   | 過去問に関する課題             |
| `type:教材`     | 教材選定・教材進行に関する課題 |
| `type:計算ミス` | 計算ミス対策                   |
| `type:計画修正` | 学習計画の見直し               |

## 優先度ラベル

| ラベル            | 用途                           |
| ----------------- | ------------------------------ |
| `priority:high`   | 直近で得点に大きく影響する課題 |
| `priority:medium` | 通常の復習・改善課題           |
| `priority:low`    | 余力があるときに扱う課題       |

## 状態ラベル

| ラベル              | 用途                               |
| ------------------- | ---------------------------------- |
| `status:要復習`     | 復習内容が未着手の課題             |
| `status:再演習待ち` | 一度対策し、再演習を待っている課題 |
| `status:完了`       | 完了済みの課題                     |
